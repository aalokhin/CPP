Здоровенькі були.

(Якщо якась із команд не виконуться через обмеження доступу викорстовуйте sudo !!!!!!!!!!!!!!!!!)
Щоб правильно зібрати проект необхідно налаштувати середовище таким чином: 

sudo apt-get update 		# ця команда обновить індекс пакетів apt-get (якшо у вас виникнуть труднощі із apt-get - google в допомогу;))
sudo apt-get upgrade		# ця команда оновить пакети (не обов'язково)

Далі необхіжо встановити компілятор GCC:

sudo apt install build-essential		# ця команда встановить GCC
gcc -v		# ця команда перевірить встановлення  gcc

Далі встановимо бібліотеку ncurses:

apt install libncurses5-dev libncursesw5-dev		# встановлення ncurses

Тепер можна перейти до компіляції проекту.
Спочатку необхідно зберегти проект в папку home до вашого apt(google=)).
Тепер необхідно перейти до каталогу з проектом:
cd 	# команда зміни робочої директорії
ls	# перегляд доступних файлів в робочій директорії
Робоча директорія повинна бути .../RUSH00/
Фінальна стадія - компіляція:
sudo make	# виконає компіляцію та збірку проекту у файл  ft_retro
./ft_retro 	# запустить гру.
Щоб завершить гру натисніть "q".
